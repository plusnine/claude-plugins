# plan.md Format

## Structure

```
# Plan — {JIRA ID or kebab-case name}

Date: {YYYY-MM-DD}
Status: DRAFT | APPROVED
```

## Task Table

```
## Tasks

| # | Task | Description | Prerequisites |
|---|------|-------------|---------------|
| 01 | {task name} | {one sentence: what this task does} | none |
| 02 | {task name} | {one sentence: what this task does} | #01 |
```

Status becomes `APPROVED` after explicit user confirmation.
